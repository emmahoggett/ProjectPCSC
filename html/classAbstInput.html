Abstract function that read the file. The function can have a Random_Variable, which give to the user the opportunity to build different type of vector distribution.

Parameters
pRandomVar: Pointer of Random variable, which gives a vector and the parameter of the vectors distribution.
alpha: The confidence interval that can be precised in the file or set as default in the children of the class.
moment: Order of the moment that can be precised in the file or set as default in the children of the class.
file_name: File name given by the user or that was set as default.
